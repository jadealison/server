La historia de HTML

HTML (Hypertext Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web
sirve de referencia en el software que conecta la elaboración de una página
define una estructura básica y un código(HTML) para la definición de contenido

Versiones de HTML:
  • Version 2.0 1995
  • Version 3.2 1997
  • Version 4.0 1999
  • Version 4.01 1999
  • Version 5.0 2012

Tim Berners-Lee(TBL) en 1991
Lee consideraba el HTML una ampliación de SGML
La publicación a mediados de 1993 por la IETF
El borrador del hypertext markup language de berners y dan connolly
incluye una definición de tipo de documento SGML para definir la gramática
